# DevelopersHub Data Science Internship Tasks

## Internship Summary

This repository contains five Data Science projects completed during my internship at DevelopersHub. The tasks were designed to provide hands-on experience with real-world datasets and the complete data science workflow, including data cleaning, exploratory data analysis (EDA), model building, evaluation, and interpretation of results.

### Projects Included

#### Task 1: Iris Flower Classification

Developed a classification model to identify iris flower species based on flower measurements. Performed data exploration, visualization, and model evaluation to achieve accurate predictions.

#### Task 2: Customer Churn Prediction

Built a predictive model to identify customers who are likely to leave a service. Conducted data preprocessing, exploratory analysis, and classification modeling to understand customer behavior and churn patterns.

#### Task 3: Credit Risk Prediction

Created a model to predict loan approval decisions using applicant information. Applied data cleaning, feature encoding, and classification techniques to assess credit risk effectively.

#### Task 4: Insurance Claim Prediction

Implemented a regression-based solution to estimate insurance claim amounts using customer and policy-related data. Evaluated model performance using appropriate regression metrics.

#### Task 5: Loan Approval Prediction

Developed a predictive model to determine whether a loan application would be approved or rejected. Performed preprocessing, feature engineering, model training, and performance evaluation.

## Approach

For each project, the following workflow was followed:

* Understanding the dataset and problem statement
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA) using visualizations
* Feature preparation and encoding
* Model training and testing
* Performance evaluation using suitable metrics
* Drawing conclusions and key insights
